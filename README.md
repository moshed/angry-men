# The Angry Men, Not Dead Yet

One question, settled by vote: **who here is actually funny?**
Fourteen men, one board each, secret ballot.

**→ [moshed.github.io/angry-men](https://moshed.github.io/angry-men/)**

Each man votes from his own private link. Anyone can read the results, and
**nobody — including whoever runs it — sees an individual board on the site.**

- **Your board** — drag all fourteen into order and lock it in. Change it as often
  as you like until the deadline; only your newest board counts.
- **Consensus** — each man's average position, plus the best and worst any single
  board gave him.
- **Positions** — how many boards put each man in each slot. Sort by a slot to see
  who collected the most of them. This is the closest anyone gets to a ballot.

Everyone ranks all fourteen, themselves included — and nobody will ever know where
you put yourself.

The 2020 boards are loaded in and set the default order. Mordy won that one at 2.44.
